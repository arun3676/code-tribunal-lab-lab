# Legacy Modules

This directory is the archive bucket for the pre-Code Council application.

It exists to preserve earlier iterations of the project as engineering evidence:

- Streamlit UI experiments
- Dashboard / persistence work
- CI/CD automation
- Framework, cloud, and container analyzers
- Historical docs and assets

These files are **not part of the current runtime**.
The active product lives in:

- `apps/web` for the Next.js frontend
- `apps/api` for the FastAPI backend

## Assets

`legacy/assets/` holds design reference images and multimodal test samples. See [`assets/README.md`](./assets/README.md).
